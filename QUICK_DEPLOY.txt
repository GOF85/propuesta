### Terminal Window 2: Restart Server via SSH

```bash
# SSH to server
ssh guiyo@188.95.113.225

# Navigate to project
cd /var/www/vhosts/micecatering.eu/propuesta.micecatering.eu

# Stop current Node process
pkill -f "node src/server.js"
sleep 2

# Start with PM2 (if available)
pm2 restart propuesta

# OR start manually
node src/server.js &

# Verify running
ps aux | grep "node"
```
